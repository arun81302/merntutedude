Compound Interest Calculator

A simple JavaScript program that calculates compound interest using static values.

Features
Calculates total amount using compound interest formula
Displays compound interest in the console
Uses basic JavaScript (no libraries required)

Formula Used
A = P × (1 + R / 100) ^ T

Where:
P = Principal amount
R = Rate of interest
T = Time (in years)


 How to Run
 Open the index.html file in your browser
 Open Developer Tools (Right-click → Inspect → Console)
 View the output in the console



Example Values
 Principal = 10000  
 Rate = 5%  
 Time = 3 years  

Output
Displays:
 Total Amount
 Compound Interest
