Laundry Wallah - Responsive Landing Page

A modern and responsive landing page for a laundry service website built using HTML5 and CSS3. This project demonstrates layout structuring, flexbox usage, and responsive design with media queries.

Features

 Clean and modern UI design
Fully responsive (Desktop, Tablet, Mobile)
Flexbox-based layout
 Mobile-first improvements
 Interactive hover effects
 Organized and reusable CSS

Technologies Used

HTML5
CSS3
Flexbox
Media Queries

 Responsive Design

The layout adapts to different screen sizes:

Desktop: Two-column layout
Mobile: Stacked layout (image below text)
Tablet: Adjusted font sizes and spacing

Learnings

Proper use of flexbox for layout
Handling image responsiveness using object-fit
Fixing overflow issues using overflow: hidden
Importance of parent container dimensions
Writing clean and scalable CSS

How to Run

Download or clone the project
Open `index.html` in your browser
