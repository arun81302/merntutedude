# 🔐 Login Page UI (Task 6)

A simple  **Login Page UI** built using HTML and CSS.  
This project demonstrates layout design using Flexbox and basic form styling.

---

## 📌 Features

- Split screen layout (Image + Login form)
- Clean and modern UI
- Styled input fields with placeholder
- Hover effects on buttons and links
- Fully centered login form
- Responsive structure (basic)

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox)

---


---

## 🚀 How to Run

1. Download or clone the repository
2. Open the folder
3. Run `index.html` in your browser

---

## 🎨 UI Sections

- **Left Side** → Image section  
- **Right Side** → Login form  
  - Welcome message  
  - Email & Password inputs  
  - Forgot password link  
  - Login button  
  - Register link
