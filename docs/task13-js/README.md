Even or Odd Checker

A simple JavaScript program that checks whether a given number is even or odd using template literals.

Features
- Checks if a number is even or odd
- Displays output with the number included
- Uses clean and modern JavaScript (template literals)

Logic
- If a number is divisible by 2 → Even
- Otherwise → Odd


 How to Run
 Open the index.html file in your browser
 Open Developer Tools (Right-click → Inspect → Console)
 View the output in the console
