📖 Description

This project is a simple webpage that displays an image with a button placed directly below it. The layout ensures there is no unwanted space between the image and the button.

🚀 Features

Responsive image (fills container width)

Button with gradient background

Clean layout with no spacing issues

Simple and beginner-friendly HTML & CSS

🛠️ Technologies Used

HTML5

CSS3 (Flexbox & Gradient)

📂 Project Structure
project-folder/
│── index.html
│── style.css
│── pexels-buxteh-32156544.jpg
▶️ How to Run

Download or clone the project

Open index.html in your browser

🎨 Styling Highlights

Used linear-gradient for button background

Removed default spacing using:

margin: 0

padding: 0

display: block for image
