🧺 Laundry Service Website

A simple landing page for a laundry service built using HTML and CSS. This project demonstrates layout design using floats, basic styling, and section structuring.

📌 Features

Header with navigation bar (Logo, Menu, Username)

Hero section with:

Catchy heading and description

Image section

Clean and minimal UI design

Hover effects for better user interaction

Technologies Used

HTML5

CSS3

How to Run

Download or clone the repository

Open the project folder

Run index.html in your browser
