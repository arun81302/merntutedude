

Objective
This project displays student records in a card format using HTML, CSS, and JavaScript. It also includes a search feature to filter students by name dynamically.

Features
- Display student details (Name, Class, Marks, Address)
- Responsive card layout
- Real-time search functionality
- Uses JavaScript map() and filter()

Technologies Used
- HTML
- CSS
- JavaScript

How It Works
1. Student data is stored in an array of objects.
2. The map() function is used to render student cards.
3. The filter() function is used to search students by name.
4. As the user types in the search box, results update instantly.
