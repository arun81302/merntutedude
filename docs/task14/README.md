Number Analysis Program (JavaScript)

A single JavaScript program that performs multiple operations on a given number.

Features
- Calculate the sum of digits
- Check if the number is an Armstrong number
- Check if the number is a Prime number
- Find all factors of the number

Concepts Used
- Loops (while, for)
- Conditional statements (if-else)
- Math functions (Math.pow, Math.sqrt)


 How to Run
 Open the index.html file in your browser
 Open Developer Tools (Right-click → Inspect → Console)
 View the output in the console
